# macOS Tips & Tricks

## 最实用
- **⇧⌘/** — 搜索当前 app 所有菜单项
- **⌘\`** — 在同一 app 窗口间切换

## 截图
- ⇧⌘4 后按 Space 选窗口
- 按住 Option 去阴影
- 按住 Control 复制到剪贴板

## 窗口管理
- Option+点击切换窗口同时隐藏前一个 app

## Finder
- ⌘⇧. 显示隐藏文件
- 拖文件到 Open/Save 对话框直接跳转

## 文本
- ⌃⌘D 悬停查词典定义
- ⌥⇧K 插入 Apple logo
